# 4.27
## 刚面对新项目
对于这个项目毫无头绪，甚至哪个文件夹是干嘛的都不知道，只记得 ` npm install ` 和 ` npm run dev ` 这两个命令。
## 4.29 
### 1. 前端（vecode）
在借助AI工具的帮助下，在shoptour-backstage、shoptour-frontend两个前端项目中分别运行了 ` npm install ` 和 ` npm run dev ` 可以正常启动，端口号为<ins> http://localhsot:8080 </ins>。
### 2. 后端（IDEA）
在进入上述端口之后是一个登录页面，这时候需要进去idea找到shoptour-platform中的web文件夹，运行三个包中的agent和app文件夹中的jar包，这时候就可以在之前的8080端口输入 <ins> http://localhsot:9088/admin/doc.html </ins> ，之后再终端输入 ` Test-NetConnection -ComputerName localhost -Port 9088 ` 测试连通性，终端输出**true**则连通成功。

如果运行app文件中的jar包，之后输入<ins>http://localhost:9090/app/doc.html</ins>。
