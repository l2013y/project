#### Day 3：MyBatis-Plus 数据库操作

**重点学习**：
- BaseMapper接口使用
- Lambda条件构造器
- Wrapper自定义查询
- 分页插件使用

**代码示例**：
```java
// 基础查询
List<User> list = userMapper.selectList(null);

// 条件查询
LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
wrapper.eq(User::getDeptId, deptId)
       .like(User::getUserName, keyword)
       .orderByDesc(User::getCreateTime);
```

#### Day 4：Controller开发

**重点学习**：
- RESTful接口设计规范
- @Validated参数校验
- 统一响应格式R<T>
- 全局异常处理

**代码示例**：
```java
@RestController
@RequestMapping("/system/user")
public class UserController {

    @GetMapping("/list")
    public R<IPage<UserVO>> list(@Validated UserQuery query) {
        return R.ok(userService.pageList(query));
    }
}
```

## 快速上手小技巧

### 1. 先看后端，再看前端
后端架构更清晰，通过Swagger文档可以快速理解API接口

### 2. 模仿学习
找一个简单模块（如字典管理 dict）完整走一遍开发流程：
- 数据库表 → Entity → Mapper → Service → Controller → 前端页面

### 3. 多用断点调试
在关键位置打断点，跟踪代码执行顺序：
- Controller接收请求
- Service处理业务
- Mapper执行SQL
- 前端渲染数据

### 4. 记笔记
建立个人知识库，记录：
- 遇到的问题
- 解决方案
- 代码片段
- 配置说明

### 5. 多问为什么
不只是复制代码，要理解：
- 为什么这样设计
- 有没有更好的方式
- 这样做有什么优缺点
