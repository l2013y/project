## 全局前置守卫
也被称为*beforeEach*钩子，在使用路由进行页面跳转时都会执行，主要用于跳转前进行用户身份验证，如果验证成功便跳转，不成功就返回到登录页面进行登录
```javascript
// 注册一个全局前置守卫
router.beforeEach((to, from, next) => {
  // 这里的代码会在每次路由跳转时自动执行
  console.log('即将从', from.path, '跳转到', to.path);
  // 检查用户是否登录
  if (to.path !== '/login' && !getToken()) {
    next('/login');  // 没登录就踢回登录页
  } else {
    next();  // 已登录，放行
  }
})
```
` to.path !== '/login' && !getToken() ` 的意思是如果用户跳转的页面不是登录页面，或者getToken（一个函数，如果用户登陆过，就会存储一个凭证，函数就能返回true，没登陆就会返回false）的返回值为false，就需要返回登录页面进行登录

### v-for / v-if 
**v-for**类似java的for循环,一般会在script标签里设置一个数据源，可以用v-for为这个数据源添加列表，表格等一系列操作
常用数组方法有slice（截取数组的一部分）、map（对每个函数执行函数，并返回新数组）、filter（筛选符合条件的元素，返回新数组）
![](/images/arr.png)
```javascript
<div>v-for ="userList.slice(0,3)"//slice是Javascript中的内置数组方法,从数组中的0开始索引到2
//取决于括号内的数字，目的是优化性能，如果数据太多，页面只需要显示slice指定的数据即可
</div>
<script>
data(){
	userList = ['张三','李四','王五'...] //假设这里有多个数据
}
</script>
```
**v-if**满足条件就显示v-if其中代码，不满足条件就相当于这行代码不存在（不会出现在页面上）例：
```javascript
<p v-if = 'true'>页面显示</p>
<p v-if = 'flase'>页面不显示</p>
<p v-if ='age>18'>成年</p>
```
